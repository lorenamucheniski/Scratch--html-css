O uso da programação em Scratch para o desenvolvimento de habilidades em crianças

Estudar lógica de programação é o passo inicial para quem deseja entrar no mundo do desenvolvimento.

Com o Scratch , é possível aproveitar a infância para desenvolver habilidades como a lógica de programação e ainda se divertir. Ele contém um tipo de programação simples, visual e que permite que qualquer um consiga criar jogos, animações e diversas histórias com facilidade. Também é possível que se trabalhe a personalização de imagens e sons, o que inclui o uso da criatividade de cada um.

Criando uma conta no Scratch

É possível baixar o Scratch desktop e armazenar os seus projetos localmente na sua máquina ou você pode criar uma conta diretamente no site, assim, você poderá compartilha-los com mais pessoas. Para criar uma nova conta, basta acessar o site e selecionar a opção "inscreva-se", como mosta a imagem abaixo:

Em seguida você precisa-rá criar um nome de usuário (a plataforma pede para que não seja o seu nome verdadeiro) que ainda não exista, e uma senha com no mínimo 6 letras ou números. Feito isso vá para o próximo passo.

No próximo passo será nescessário apenas selecionar o país em que você reside.
